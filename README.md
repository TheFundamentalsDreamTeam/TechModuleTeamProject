### Problem 1.                 
**Debit Card Number**
***(C# project)***

It is a program, which receives **4** **integers** on the
console and **prints them** in **4-digit debit card format**. See the
examples below for the appropriate formatting.

#### Examples
====================
***Input :***
12
433 
1
5331
***Output :***
0012 0433 0001 5331
====================
***Input :***
9182
4221
12
3	
***Output :***
9182 4221 0012 0003
====================
***Input :***
812
321
123
22	
***Output :***
0812 0321 0123 0022
====================
